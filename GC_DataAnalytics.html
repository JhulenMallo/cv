Resumen

Este programa de certificación está diseñado para preparar a los estudiantes para roles como analista de datos en la nube, técnico de datos cloud y analista de operaciones cloud.

Los analistas de datos en la nube aprovechan la computación en la nube para preparar, procesar y analizar datos que apoyan decisiones empresariales clave. Al finalizar el programa, los estudiantes pueden compartir hallazgos de manera efectiva con stakeholders y proporcionar recomendaciones basadas en datos.

Habilidades Desarrolladas:
BigQuery Business Intelligence Cloud Computing Data Analysis Data Analytics Data Modeling Data Transformation Data Visualization Google Cloud Project Management

Cursos del Programa (5 Cursos)

1
Introduction to Data Analytics in Google Cloud

Fundamentos de análisis de datos en Google Cloud. Introducción a las herramientas y servicios esenciales para trabajar con datos en la nube.

2
Data Management and Storage in the Cloud

Gestión y almacenamiento de datos en entornos cloud. Estrategias para organizar, almacenar y acceder a datos de manera eficiente.

3
Data Transformation in the Cloud

Técnicas de transformación de datos utilizando herramientas cloud. Limpieza, procesamiento y preparación de datos para análisis.

4
The Power of Storytelling: How to Visualize Data in the Cloud

Visualización de datos y storytelling. Creación de dashboards y presentaciones efectivas para comunicar insights de datos.

5
Put It All Together: Prepare for a Cloud Data Analyst Job

Proyecto integrador final que consolida todo el aprendizaje del programa. Preparación profesional para roles de analista de datos en la nube, incluyendo mejores prácticas y técnicas de presentación de resultados a stakeholders.
