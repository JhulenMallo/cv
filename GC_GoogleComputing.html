Google Cloud Computing Foundations Certificate

Google Cloud

Programa de certificación que ayuda a desarrollar competencia técnica en computación en la nube. Este certificado es una base ideal para diversas trayectorias profesionales, incluyendo infraestructura IT, ingeniería cloud y desarrollo de aplicaciones cloud-native.

Julio 2024
4 Cursos + 4 Skill Badges
Verificado

Resumen

Este programa integral está diseñado para desarrollar competencias técnicas fundamentales en computación en la nube utilizando Google Cloud Platform.

El certificado proporciona una base sólida para múltiples trayectorias profesionales en tecnología cloud, incluyendo roles en infraestructura IT, ingeniería cloud y desarrollo de aplicaciones nativas de la nube. Combina conocimientos teóricos con habilidades prácticas verificadas mediante skill badges.

Habilidades Técnicas:
APIs AppDev BigQuery Cloud Computing Cloud Functions Cloud Infrastructure Cloud Storage Compute Engine Data & ML DataProc IAM Network Security

Cursos Fundamentales (4 Cursos)

1
Cloud Computing Fundamentals

Introducción a los conceptos fundamentales de computación en la nube, arquitecturas cloud y servicios básicos de Google Cloud Platform.

Cloud Foundations
2
Infrastructure in Google Cloud

Diseño y gestión de infraestructura en Google Cloud, incluyendo Compute Engine, almacenamiento y servicios de red.

Infrastructure Storage
3
Networking & Security in Google Cloud

Configuración de redes virtuales, balanceadores de carga, y implementación de medidas de seguridad y control de acceso (IAM).

Networking Security
4
Data, ML, and AI in Google Cloud

Servicios de datos y machine learning en Google Cloud, incluyendo BigQuery, DataProc y APIs de ML/AI.

Data ML & AI

Skill Badges Prácticos (4 Badges)

Además de los cursos teóricos, este programa incluye 4 skill badges que validan habilidades prácticas mediante laboratorios hands-on:

Implement Load Balancing on Compute Engine

Implementación de balanceadores de carga para distribuir tráfico entre instancias de Compute Engine.

Set Up an App Dev Environment on Google Cloud

Configuración de entornos de desarrollo para aplicaciones utilizando servicios de Google Cloud.

Build a Secure Google Cloud Network

Construcción de redes seguras en Google Cloud con firewalls, VPCs y políticas de seguridad.

Prepare Data for ML APIs on Google Cloud

Preparación y procesamiento de datos para utilizar APIs de Machine Learning de Google Cloud.

Trayectorias Profesionales

Este certificado prepara para múltiples roles en tecnología cloud:

IT Infrastructure Specialist

Diseñar, implementar y mantener infraestructuras de TI en entornos cloud.

Cloud Engineer

Desarrollar e implementar soluciones utilizando servicios de Google Cloud Platform.

Cloud-Native Developer

Crear aplicaciones nativas de la nube usando arquitecturas modernas y escalables.
