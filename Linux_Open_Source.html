Resumen

Esta especialización de The Linux Foundation permite comprender el ecosistema open source y dominar herramientas esenciales del entorno Linux para desarrolladores. También enseña el uso de Git y GitHub para trabajo colaborativo en proyectos distribuidos.

A través de los cursos se adquieren habilidades prácticas en scripting, manejo de versiones, automatización de tareas y buenas prácticas en comunidades open source.

Habilidades Clave:
Linux Bash Shell Scripting Git GitHub CI/CD Open Source Command-Line Interface System Configuration

Cursos de la Especialización (4 Cursos)

Open Source Software Development Methods

Introducción a las comunidades open source, licencias, procesos de contribución y flujos de trabajo colaborativos.

3 semanas • 10-15 horas • Calificación: 100%

Linux for Developers

Uso de la línea de comandos, gestión de archivos, permisos, procesos y herramientas esenciales de Linux.

4 semanas • 15-20 horas • Calificación: 100%

Linux Tools for Developers

Uso de compiladores, depuradores, scripts y herramientas CLI avanzadas en entornos Linux.

4 semanas • 15-20 horas • Calificación: 100%

Using Git for Distributed Development

Dominio del control de versiones distribuido con Git y GitHub, ramas, fusiones y trabajo colaborativo.

3 semanas • 10-15 horas • Calificación: 100%
